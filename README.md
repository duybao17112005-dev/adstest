# GreenNest — Sustainable Home Goods Website

GreenNest is a responsive sustainability-focused eCommerce landing page built with pure HTML, CSS, and JavaScript.

The project showcases eco-friendly home goods, ethical sourcing standards, and environmental impact transparency through a modern multi-section website experience.

## Features

- Fully responsive mobile-first layout
- Modern UI/UX design
- Multi-page SPA navigation
- Product showcase sections
- Sustainability impact dashboard
- Earth Standard certification framework
- Interactive mobile navigation
- Animated live CO₂ counter
- SEO-ready meta tags
- GitHub Pages deployment

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Responsive Grid/Flexbox
- GitHub Pages

## Pages Included

### Home
Main landing page introducing the GreenNest brand and featured products.

### Shop
Product catalog with category filters and sorting options.

### Earth Standard
Transparency page explaining sustainability certifications and sourcing criteria.

### Impact
Environmental impact statistics and sustainability timeline.

### Journal
Educational sustainability blog and buying guides.
